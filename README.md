# Documentation officielle de la webradio Riff

Sources de la documentation technique officielle pour l'infrastructure logicielle et l'API de la webradio [Riff](https://www.riff-radio.org).

## Quick Start

Le projet est conçu pour MkDocs avec le framework Material.

- **Prérequis :** MkDocs, MkDocs-Material
- **Build :** `pip install mkdocs-material`, clôner le projet, puis `mkdocs build`

## Architecture

- `docs/` : Contient le contenu textuel (format Markdown), le CSS et les images.
- `mkdocs.yml` : Configuration du moteur pour le projet.
